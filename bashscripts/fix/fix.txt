## üîê Configurazione SSH Completa

### Setup SSH Agent e Chiavi
```bash
# Avvia SSH Agent
eval "$(ssh-agent -s)"

# Genera nuova chiave SSH (ED25519 - pi√π sicura)
ssh-keygen -t ed25519 -C "email@example.com"

# Oppure RSA (se ED25519 non supportata)
ssh-keygen -t rsa -b 4096 -C "email@example.com"

# Aggiungi chiave all'agent
ssh-add ~/.ssh/id_ed25519

# Verifica chiavi caricate
ssh-add -L

# Testa connessione GitHub
ssh -T git@github.com

# Copia chiave pubblica negli appunti (Ubuntu)
cat ~/.ssh/id_ed25519.pub | xclip -selection clipboard
```

### Configurazione SSH Config
```bash
# Crea/modifica ~/.ssh/config

# Esempio configurazione:
# Host github.com
#   HostName github.com
#   User git
#   IdentityFile ~/.ssh/id_ed25519

## üîç Ricerca e Navigazione File

### Comandi Grep Avanzati
```bash
# Cerca testo ricorsivamente (ignora case)
grep -Ril "text-to-find" .

# Cerca con esclusioni
grep -Ril "text" . --exclude-dir=node_modules --exclude-dir=.git

# Cerca solo in file specifici
grep -Ril "text" . --include="*.php" --include="*.js"

# Conta occorrenze
grep -c "pattern" file.txt

# Mostra righe prima e dopo
grep -A 3 -B 3 "pattern" file.txt
```

### Find e Locate
```bash
# Trova file per nome
find . -name "*.php" -type f

# Trova file modificati negli ultimi 7 giorni
find . -mtime -7 -type f

# Trova directory vuote
find . -type d -empty

# Trova file pi√π grandi di 100MB
find . -size +100M -type f
```

## üßπ Pulizia e Manutenzione

### Rimozione File Temporanei
```bash
# Rimuovi file Zone.Identifier (Windows)
find . -type f -name "*:Zone.Identifier" -delete

# Rimuovi file temporanei comuni
find . -name "*.tmp" -delete
find . -name "*.log" -delete
find . -name ".DS_Store" -delete

# Pulizia Git
git clean -fd  # Rimuove file non tracciati
git gc --prune=now  # Garbage collection
```

### Gestione Permessi Sicura
```bash
# Permessi standard per progetti web
find . -type d -exec chmod 755 {} \;  # Directory
find . -type f -exec chmod 644 {} \;  # File

# Solo script eseguibili
chmod +x script.sh

# Aggiungi utente al gruppo www-data
sudo usermod -aG www-data $(whoami)

# Cambia proprietario (solo se necessario)
sudo chown -R $(whoami):www-data .
```

## ‚ö†Ô∏è Risoluzione Problemi Comuni

### Permission Denied
```bash
# Problema: zsh: permission denied: ./script.sh
# Soluzione:
chmod +x script.sh
```

### Include Path Errato (PHP)
```bash
# Problema: include(/path/to/file): Failed to open stream
# Soluzioni:
1. Controlla path relativi/assoluti
2. Verifica permessi directory
3. Rimuovi suffisso "_old" da cartelle moduli
4. Controlla include_path in php.ini
```

### Git Push Rejected
```bash
# Problema: Updates were rejected
# Soluzioni:
git pull --rebase origin main
# Oppure (se sicuri):
git push --force-with-lease origin main
```

### SSH Connection Issues
```bash
# Problema: Permission denied (publickey)
# Soluzioni:
ssh-add ~/.ssh/id_ed25519
ssh -vT git@github.com  # Debug verboso
# Verifica che la chiave sia aggiunta al servizio Git
```

### Merge Conflicts
```bash
# Risolvi conflitti
git status  # Vedi file in conflitto
# Modifica manualmente i file
git add <file-risolto>
git commit

# Oppure usa tool grafico
git mergetool
```
